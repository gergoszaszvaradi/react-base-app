# React Base App
This repository contains react web application boilerplate projects.

There are multiple configurations, all of them separated into branches.
## Using the repository
Specifying a configuration can be done by cloning the repository with the right branch
```bash
git clone https://github.com/gergoszaszvaradi/react-base-app -b BRANCH_NAME
```
Once the repository is downloaded, remove the `.git` folder to unlink the project from this repository, then initialize a new git repository

The next step is to run the configuration script
```bash
npm run configure
```
```bash
yarn configure
```
Then install the npm packages
```bash
npm install
```
```bash
yarn install
```
## Usage
Running the dev server
```bash
npm run start
```
```bash
yarn start
```
Building the project
```bash
npm run build
```
```bash
yarn build
```
